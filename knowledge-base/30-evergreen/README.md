# Evergreen Notes - 永久笔记

一个想法，一个笔记。长期有效，持续更新。

## 什么是永久笔记

永久笔记（Evergreen Notes）是：
- **原子化**: 一个笔记只包含一个核心概念
- **永久有效**: 不随时间过期，持续更新
- **高度链接**: 与其他笔记建立大量双向链接
- **概念导向**: 关注概念而非项目或时间

## 与项目笔记的区别

| 维度 | 永久笔记 | 项目笔记 |
|------|----------|----------|
| 时间 | 永久有效 | 有开始和结束 |
| 范围 | 单一概念 | 多概念组合 |
| 更新 | 持续迭代 | 完成后归档 |
| 链接 | 高度互联 | 项目内链接 |
