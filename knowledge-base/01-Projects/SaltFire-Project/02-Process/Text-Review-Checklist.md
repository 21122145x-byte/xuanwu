# 《盐火预言》文本审校清单 (Text Review Checklist)

本清单用于对《盐火预言》的初稿及修订稿进行结构化验证，确保文本质量符合“盐火写作工坊”的标准。审校维度分为：逻辑、设定、文笔。

## 一、逻辑 (Logic)
**核心目标**：确保故事立得住，因果咬合紧密，人物行为有根基。

- [ ] **动机校验**：人物的每一个关键抉择和行动，是否有足够、且符合其性格的动机支撑？（避免“工具人”式的强行推动）
- [ ] **因果连贯**：情节的发展是否符合严密的因果律？（避免过度依赖巧合或“机械降神”解决冲突）
- [ ] **状态一致性**：检查前后文的时间线、空间位置、物品状态（如：上一场戏受伤，下一场戏是否还带着伤？）、人物知情状态是否存在矛盾？
- [ ] **核心推演**：本章剧情是否在真正推进“战争如何先在叙事和系统中被写出来”这一核心命题？

## 二、设定 (Worldbuilding)
**核心目标**：确保世界观是故事的土壤，而不是阻碍阅读的说明书。

- [ ] **展现大于讲述 (Show, Don't Tell)**：设定是否通过人物动作、环境细节、冲突自然流露？（警惕并标红大段的“百科全书式”生硬解说）
- [ ] **规则自洽**：故事中的事件和人物能力，是否严格遵守了已确立的世界观规则？（如：炽盐的特性、天幔的机制、三大势力的基本制衡）
- [ ] **认知负担**：单章或单场戏中，是否一次性抛出了过多新名词/新概念？（如有，需建议打散或延后释放）
- [ ] **设定功能性**：出现的设定是否对当前人物困境或情节推进有实质作用？（剔除仅仅为了“看起来很酷”但冗余的无效设定）

## 三、文笔与表现 (Writing Style & Execution)
**核心目标**：强化“人味”与“绷劲”，提供清晰的阅读质感。

- [ ] **视点(POV)稳定**：本场戏的叙事视点是否清晰统一？（是否存在越界描写了当前视点人物不可能看到、听到、想到的东西？）
- [ ] **台词人味**：人物对话是否符合其阶级、立场、情绪和性格？（避免不同人物说出同一套句式，避免过度“舞台剧腔”或说教）
- [ ] **画面与五感**：场景描写是否调用了视觉之外的感官（听觉、嗅觉、触觉等）？是否有建立起冷酷、燥热、压抑等核心氛围的真实细节？
- [ ] **节奏与绷劲**：
    - 紧张桥段：是否使用了较短的句式、干净的动词？
    - 情绪停顿：是否给了人物反应的空间？
    - **冗余清理**：标出并建议删除多余的形容词、副词、以及无意义的心理过渡词（如“他想”、“她觉得”）。
