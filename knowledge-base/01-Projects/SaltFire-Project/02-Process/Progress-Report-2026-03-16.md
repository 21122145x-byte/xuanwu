### 3.2 节奏模型（Shaun提出）

```
平静水池 → 落叶微澜 → 涟漪 → 鱼儿翻身 → 风大 → 欢腾 → 暴雨

对应：
- 岗位麻木（平静）
- 异常初现（微澜）
- 意识到不对（涟漪）
- 身体反应（翻身）
- 外部压力（风大）
- 拆校验发现尾标（欢腾）
- "这条预警不干净"（暴雨）
```

### 3.3 修订原则

> **人味不能把刀口泡软，刀口也不能把人写成工具。**
> 
> **先让人进戏，再让世界漏出来；先让人物承受，再让主题浮上来。**
> 
> **把设定往后藏，把事件往前顶，把人先钉在火上。**
