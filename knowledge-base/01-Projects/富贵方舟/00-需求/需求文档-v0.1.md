## 1. 原始需求（动机）

**Shaun 为什么要做这个项目？**（待确认，以下为反推）

- 想要一个个人 A股分析工具
- 目前缺乏系统性个股分析能力
- 希望把零散的股票信息整合起来
- 需要每日/每周自动跟踪

---

## 2. 核心需求

### 2.1 功能性需求

| ID | 需求描述 | 优先级 | 备注 |
|----|----------|--------|------|
| F01 | 个股与大盘关联分析 | P0 | 贝塔系数、相关性、板块归属、资金流向 |
| F02 | 企业基本面信息整合 | P0 | 公司概况、财务健康度、股东结构 |
| F03 | 企业公告/事件监控 | P1 | 定期报告、重大事项、股权变动、监管动态 |
| F04 | 政策解读 | P1 | 宏观、行业、区域政策对个股的影响 |
| F05 | 历史数据对照 | P1 | 季节性规律、事件驱动、技术形态 |
| F06 | 走势预测（辅助） | P2 | 技术面+基本面+情绪面+政策面综合 |
| F07 | 个股日报自动生成 | P0 | 每日定时推送 |
| F08 | 个股周报自动生成 | P1 | 每周定时推送 |
| F09 | 事件快评 | P1 | 重大公告/政策发布后自动触发 |
| F10 | 关注股票池管理 | P0 | 可增删股票，可设置提醒条件 |

### 2.2 非功能性需求

| ID | 需求描述 | 优先级 | 备注 |
|----|----------|--------|------|
| N01 | 数据可靠性验证 | P0 | 关键数据需标注来源和时间 |
| N02 | 分析结论可解释 | P1 | 不给黑箱结论，要能说明原因 |
| N03 | 风险提示完整 | P0 | 每次推送必须包含风险提示 |
| N04 | 交付物格式规范 | P1 | 统一模板，便于阅读 |
