## 一、项目定位

富贵财经当前定位为：
- 面向 A 股市场的日报 / 午盘 / 收盘 / 周回顾 / 周展望生产项目
- 目标不是“写一篇财经稿”，而是形成一套**连起来的自动化链路**
- 当前主攻方向是：
  1. 午盘版自动初稿基线稳定
  2. 评估者 subagent 接管数据校验与初稿审计
  3. Anna 负责维护评估者机制并审计终稿

## 四、午盘版全自动流程图（主链）

当前正式口径：

```text
S1 数据拉取
  ↓
S2 数据清洗
  ↓
J4a-1 数据校验（评估者 subagent）
  ↓
S4 初稿生成
  ↓
J4a-2 初稿审计（评估者 subagent）
  ↓
S5 文字终稿生成
  ↓
J4b 终稿审计（Anna）
  ↓
放行 / 打回 / 暂停
```

说明：
- 不再拆成两条断开的链
- 数据校验和初稿审计都前置给评估者 subagent
- Anna 不做中间补洞，负责维护评估者并审终稿
