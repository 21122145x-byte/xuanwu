## 五、目标 skill 拆分

建议拆成 5 个 skill：

### 1. fugui-finance-collector
职责：
- 拉取指数 / 板块 / 涨停池 / 资金流向 / breadth
- 处理主源 / 备用源 / 降级路径

来源脚本：
- `data_collector_skill.py`

### 2. fugui-finance-processor
职责：
- 清洗结构化数据
- 保留写稿与评估关键字段

来源脚本：
- `data_processor_skill.py`

### 3. fugui-finance-validator
职责：
- 数据校验（J4a-1）
- 后续也可扩展为初稿校验基础层

来源脚本：
- `data_validator_skill.py`

### 4. fugui-finance-writer
职责：
- 生成午盘初稿
- 后续扩展到收盘 / 周回顾 / 周展望

来源脚本：
- `writer_skill.py`

### 5. fugui-finance-pipeline
职责：
- 串联 collector / processor / validator / writer
- 后续接入评估者 subagent 和终稿层

来源脚本：
- `run_pipeline.py`
- `run_pipeline_local.py`
