# 万界碑｜第一章试写片段 v1

矿里的夜和白天没什么分别。

灯火常年吊在头顶，油烟熏得发黑，照下来也只是把石壁上的水痕和人脸上的灰照得更脏一点。风从坑道深处一阵阵往外涌，带着潮气、铁锈味，还有一种说不清的腥，像有东西烂在更下面，一直没挖出来。

陆沉蹲在碎石堆边，把最后半口发硬的饼塞进嘴里，没怎么嚼就咽了下去。喉咙被磨得发疼，他也没在意，只低头用指甲把掌心那道新裂开的口子抠干净。矿灰钻进肉里，疼得发木，血倒流得不多。

旁边有人骂了一句娘，说今天又少了一个。

少的是个老矿奴，姓什么没人真记得，只知道腿早些年被砸坏了，走路总像拖着半截烂木头。白天交班的时候人还在，晚上再点火头，铺位就空了。没人问他去哪儿了，也没人真觉得他还能回来。矿坑这地方，活人少一个，和少一筐碎石差不了太多。

“听见没有？”

说话的是瘦猴，蹲在陆沉对面，眼白被灯火一照，显得有点发黄。

“听见什么？”陆沉问。

瘦猴没立刻答，只抬头往坑道更深处看了一眼，声音压得很低：“底下刚才响了。”

陆沉手上动作停了停。

矿坑里什么声音都有。

镐子砸石头，铁链拖地，木架开裂，死人临咽气前喉咙里那两声倒气，哪样都不新鲜。可瘦猴说“响了”的时候，脸色不太对，像不是听见了什么常有的动静，而是听见了某种不该出现在这里的东西。

“什么声？”

“说不上来。”瘦猴舔了舔干裂的嘴唇，“不像塌，不像裂，也不像风。像……像有人拿铁在石头底下敲了一下。”

陆沉没说话。

坑道深处的风又涌上来一阵，比刚才更冷，带得头顶那几盏油灯都轻轻晃了晃。灯影一摇，石壁上的黑色水痕跟着颤，像有一层薄薄的东西贴在上面，正顺着岩缝往外爬。

有人不耐烦地骂：“别他妈自己吓自己，这地方哪天不响？”

话音刚落，脚下忽然传来一声闷震。

不是很大，却像从极深的地方顶上来，先是脚底一麻，紧接着整段坑道都跟着轻轻一颤。碎石从石壁上簌簌落下，灯火晃得更厉害了些，最里面那盏甚至啪地灭了，黑暗一下塌出一块口子。

这次没人再说是错觉。

坑道里安静了两息，接着有人猛地站起来，骂声、脚步声、铁链碰撞声一下全乱了。

陆沉也跟着起身，抬头朝黑下去的那一截看过去。

他不知道为什么，心口忽然缩了一下。

不是因为怕塌。

塌方这种事他见得太多了，真要死，反而不会先有这种感觉。

那更像是一种很旧、很冷的东西，在黑里轻轻醒了一下。

下一瞬，最深处的黑暗里，传来“当”的一声。

声音不大，却清得古怪。

像真有什么东西，被谁在石头底下敲了一记。
