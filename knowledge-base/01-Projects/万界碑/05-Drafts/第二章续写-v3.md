# 万界碑｜第二章续写 v3

陆沉盯着那东西，喉咙发紧，半天没动。

不是他胆子小。

是在这鬼地方，越不该动的东西，越像会要人命。

矿场里干久了，谁都知道一个死理：石头裂了可以挖，地层松了可以避，唯独那些“看着不像矿、摸着不像石、偏偏埋在最深处”的东西，最好连看都别多看一眼。

因为那往往不是财路。

是祸。

陆沉压着呼吸，强行把视线从那片冷硬的立面上挪开，转头去看四周。

烟尘比刚才又淡了一层。

这地方终于慢慢显出了真正模样。

不是坑道。

至少不全是。

头顶斜斜压着断开的木梁和塌碎的石层，把上面的路堵了大半；左边是碎石堆和崩开的裂壁，裂缝深得看不见底；右边却比想象中空，像原本就有一段更宽的地下石室，只是常年埋在矿层更深处，从没人真正挖到过。

地上散着不少东西。

断掉的铁钎，裂开的矿篓，半截发黑的木桩，甚至还有几块不像这条矿脉该有的旧石砖，边角齐整，被土灰埋得只露出一点棱。

陆沉眼皮微微一跳。

这里不是单纯塌出来的坑。

像是塌进了某个更早就埋在地下的旧地方。

这念头刚一冒出来，他心里那股凉意就更重了。

矿坑底下埋旧洞，不是什么好事。

埋得越深，说明越早；越早，也往往越不该被今天的人掀出来。

上头忽然又传来一阵闷响。

轰。

土石簌簌往下落，几粒碎石砸在陆沉肩头，疼得他立刻缩了下脖子。

他一下清醒过来。

先活。

别管那是什么，先找能出去的路。

陆沉咬了咬牙，扶着石壁，绕着那片立着的暗影往右边挪。脚下全是碎石，稍不留神就打滑，他只能一只手贴着壁，一只手护着肋下，慢慢往前探。

那东西离他并不算远。

哪怕他已经有意绕开，经过旁边时，掌心还是能感觉到那股透过空气渗出来的冷。

不是风吹的冷。

更像一口深井开在地下，井口就立在那儿，里头的寒气一层层往外漫。

陆沉没敢再碰它。

可就在他将要从那东西旁边绕过去的时候，脚下忽然踩到了什么。

咔。

像薄壳裂开的声响。

陆沉心里一紧，低头去看。

脚边半埋在土里的，不是石头。

是一截骨头。

灰白，发脆，外面裹着陈年泥垢，被他这一脚踩裂了一半。

陆沉后背一下麻了。

他顺着那截骨头旁边扒了两下，指尖很快又碰到第二截、第三截。

全都埋得不深。

像这里原本就躺过人。

不，不止一个。

陆沉呼吸微乱，强逼自己停手。

继续扒下去没有意义。

死人不会告诉他路怎么走。

可这些骨头摆在这儿，本身已经够说明问题——这个地方，之前有人来过；而且那些人，多半没能活着出去。

他抬头又看了一眼那块立着的东西，心口狠狠沉了一下。

这地方要么是旧矿坑更深处，要么干脆就是矿场一直在避、一直在压的禁地。

无论哪种，对他都一样糟。

陆沉忍着肩背的疼，继续往右摸。

右边果然比别处更空，石壁也没有左边那么乱。只是越往里走，空气里的那股味道就越怪。

不只是潮，不只是腐。

里面还混着一点说不出的铁腥味，像很多年前这里流过血，后来血干了、烂了、渗进石缝里，到今天都没散干净。

陆沉走出不到十步，脚下忽然一绊，差点又摔。

他低头一摸，摸到一根横在地上的粗铁链。

链子一半埋在土里，一半露在外面，锈得很厉害，表面却比普通铁器更沉、更硬，链环上还压着一层暗红近黑的痕，像锈，又像别的东西。

陆沉顺着铁链往前摸，越摸心里越发沉。

那链子竟是一路从右边墙角延过去，另一头，正没进那片立着的暗影底下。

像曾经有人想把那东西锁死在这里。

陆沉脑子里“走”这个念头，第一次真正被压住了。

如果只是埋了一块古怪石头，不至于用这么粗的链子。

如果只是寻常矿坑塌方，更不可能在底下锁着这种东西。

他站在原地，指节慢慢攥紧，掌心全是冷汗。

走，还是看。

往常这种时候，他不会犹豫。

看不懂的东西别碰，离得越远越好。

可现在不一样。

他已经掉下来了。

上头堵死，旁边埋骨，出口不知在哪，而这地方最像“答案”的，偏偏就是眼前这个最不该碰的东西。

陆沉牙关一点点咬紧。

他不想承认，可心里其实已经很明白——想从这鬼地方活着出去，绕开它未必有路；反倒是它，很可能知道这里为什么会是这样。

这个念头一出来，他自己都觉得荒唐。

一块不知道埋了多少年的死东西，能知道什么？

可紧接着，他耳边又响起了那一声。

当。

极轻。

却比刚才更清楚。

这一次，陆沉甚至能分出方向。

不是从四面八方传来。

就是从那东西里面。

像冷硬的壳子里，隔着很厚很厚的岁月，有什么又轻轻碰了一下。

陆沉浑身寒毛都竖起来了。

他脚下本能地想退，退了半步，却又硬生生停住。

不能一直怕。

怕没用。

矿坑塌的时候，怕没用；执鞭人抽下来的时候，怕没用；现在掉进这种地方，怕一样没用。

陆沉站在原地，盯着那片暗影看了很久，终于一点点挪了回去。

这回他走得很慢。

不是因为腿软。

是像在朝一口不知道深浅的井边挪，每一步都得先看清脚下，生怕再把自己送进更深的地方。

等走近了，烟尘已经散得差不多了。

那东西的样子，总算比先前更清楚了些。

确实像碑。

不是墓碑那种窄长薄碑。

而是一块极厚、极沉的黑碑，像从整块山体里直接截下来，再硬生生立进地下。碑身大半还埋在土石里，只露出半截，可只是这半截，已经比陆沉高出一头。

碑面上糊满了泥垢、潮锈和岁月积出来的死皮，颜色黑得发沉，不像普通石，倒像某种烧过又冷透的铁。

而在碑身偏下的位置，斜斜缠着一截断链。

链子果然是锁它的。

只是如今链环崩开了大半，像不是被人解开的，倒像有什么东西曾经从里面硬生生挣过一下，把它震裂了。

陆沉看得头皮发麻。

他不敢再往碑顶看，视线只落在掌心之前摸到的位置，慢慢抬起手，隔了半寸，停住。

碰，还是不碰。

这一下停得极久。

最后，他还是把手落了上去。

冷意瞬间顺着掌骨往里钻。

陆沉牙一紧，没缩。

他忍着那股冷，掌心一点点往旁边擦。

泥垢簌簌往下掉。

粗粝表面下，慢慢露出一道更深的痕。

不是裂纹。

是字。

或者说，是某种曾经像字的东西。

那笔画太古，太沉，根本不像如今下界通行的文字，很多转折处都像刀斧直接凿进石里，透着一种说不出的生硬和凶气。

陆沉一个也认不出。

可就在他盯着那几道模糊刻痕看的时候，脑子里却忽然像被什么东西撞了一下。

不是声音。

也不是人话。

更像一股极冷、极旧的意念，隔着不知多少年岁月，从碑里溢出来，擦着他的脑海过去。

只有短短一下。

可那一下里，陆沉像看见了什么。

不是完整画面。

只是几个极碎的残片——

天是裂的。

地上全是倒着的碑影。

有人在很远很远的地方咳血，血一直流，一直流，把脚下的石阶染得像黑夜一样。

再下一瞬，所有残片全都碎了。

陆沉眼前一黑，整个人猛地往后踉跄了两步，后腰重重撞上断岩，疼得他差点一口气没提上来。

他大口喘着气，脸色一下白了。

刚才那是什么？

幻觉？

还是这碑里真藏着什么？

陆沉死死盯着它，胸口起伏得厉害。

而那碑，依旧立在原地，黑，冷，沉，一动不动。

像什么都没发生过。

可陆沉知道，不一样了。

就在他掌心落上去、看见那几道古怪刻痕的时候，某种原本隔在他和这东西之间的东西，被碰开了一条缝。

他还说不上那是什么。

可他已经再没法把它当成一块单纯的死碑。

上头忽然又传来一阵更剧烈的塌响。

轰隆——

这一次，不只是碎石下落。

整片石室都跟着震了一下。

右边原本还算稳的石壁，忽然裂开了一道细缝。灰尘扑簌直落，刚才那条半埋在地里的粗铁链也被震得猛地一颤，发出一连串低哑的碰撞声。

哐。哐。哐。

而在这串铁响尽头，那块黑碑深处，像终于有东西被彻底惊醒。

一道比刚才清楚得多的叩声，从碑内缓缓传了出来。

当。

当。

当。

不急，不重。

却一下比一下更像某种活着的东西，在漫长死寂之后，重新确认自己还在。

陆沉脸色彻底变了。

因为第三声落下的时候，他终于听清了一点别的。

那不是单纯的叩响。

那声音后面，像还拖着半截极低、极哑的摩擦声。

像有人隔着碑，在里面，缓缓转了一下头。
