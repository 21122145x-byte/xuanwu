# 万界碑｜第一章试写片段 v3

矿里的夜和白天没什么分别。

几盏油灯钩在坑道顶上，灯罩和上面的岩层早被烟火熏得发乌，火光一晃，黑油似的污痕便沿着石缝往下淌，和渗出来的潮水混在一起，挂在石壁上，像旧伤口里迟迟渗不净的脓血。风从坑道更深处一阵阵往上顶，带着潮气、铁锈味，还有股闷了太久才会有的腥臭，像霉烂的皮肉、馊掉的汤水和死老鼠骨头一起封在石头底下，时间久了，味儿就慢慢从缝里挤出来。

陆沉蹲在碎石堆边，和旁边几个矿奴一样，趁着歇气的空档往嘴里塞东西。有人啃发硬的杂粮饼，有人捧着半碗凉得起皮的稀糊糊，还有人干脆靠在石壁上喘，懒得动。靠近里面那一排铺位空了一块，褥子卷得乱七八糟，像是人刚被拖走，连收拾都省了。

陆沉把最后半口硬得发木的饼塞进嘴里，没怎么嚼就咽了下去。那股馊酸的面味混着坑里的腥臭一起往上翻，喉咙被刮得生疼，他也没管，只低头用指甲把掌心那道刚裂开的口子抠干净。矿灰钻进肉里，疼得发木，血倒流得不多。这样的口子每天都有，好像这矿坑总得先在人身上讨一点什么，才肯让他们继续往下挖。

“又少了一个。”

石壁边有人低低骂了句娘，声音哑得像砂纸在磨。

没人接话。

火光底下，几个矿奴还是低头吃自己的。只有一个人朝那块空铺位瞥了一眼，很快又把目光挪开，像多看一眼，今天夜里被拖走的人就会轮到自己。

少的是个老矿奴，腿早些年被砸坏了，走路总像拖着半截烂木头。白天交班的时候人还在，晚上歇气再回来，那块铺位就空了。没人问他去了哪儿，也没人真觉得他还能回来。矿坑这地方，活人少一个，和少一筐碎石差不了太多，连名字都留不住。

“听见没有？”

声音压得很低，像怕惊动什么。

陆沉抬起头。

说话的是瘦猴，蹲在他对面，手里那半块饼捏了半天也没往嘴里送，眼白被火光一照，泛着浑黄。他没看陆沉，反而一直盯着坑道更深处那团照不进去的黑。

“听见什么？”陆沉问。

瘦猴舔了舔干裂的嘴唇，这才慢慢把眼珠转过来一点：“底下刚才响了。”

陆沉手上的动作停了停。

矿坑里什么声音都有。

镐子砸石头，铁链拖地，石车翻在坡道上，木架开裂，死人临咽气前喉咙里那两声倒气，哪样都不新鲜。可瘦猴脸上的神色不对，像他听见的不是塌方，不是裂岩，也不是哪截木架又撑不住了，而是什么本不该埋在这里的东西，在更深的地方，轻轻碰了一下。

“什么声？”

“说不上来。”瘦猴喉结滚了滚，声音压得更低，“不像塌，不像裂，也不像风。像……像有人隔着很厚的石头，从底下轻轻敲了一下。”

陆沉没说话。

坑道深处的风又涌上来一阵，比刚才更冷，带得头顶几盏油灯都轻轻晃了一下。灯影摇开，石壁上的污痕和水痕跟着颤，像有细长的黑影贴在岩层里，顺着裂缝一点点往上爬。陆沉抬眼看过去，心口莫名缩了一下。

不是怕塌。

塌方这种事他见得太多了，真要死，反而不会先有这种感觉。

那更像是……这矿坑底下压着的，从来就不只是矿。

有人不耐烦地骂：“少他妈自己吓自己，这地方哪天不响？”

话音刚落，脚下忽然传来一声闷震。

不是很大，却像从极深的地方顶上来，先是脚底一麻，紧接着整段坑道都轻轻一颤。碎石簌簌往下掉，木架发出短促的呻吟，最里面那盏油灯啪地一声灭了，黑暗顿时在坑道深处塌出一块口子。

这次没人再说是错觉。

坑道里安静了两息，接着骂声、脚步声、铁链碰撞声一下全乱了。有人站起来往外退，有人攥着镐柄不放，像握紧了那截破铁，命就还能留在自己手里。

陆沉也跟着起身，抬头朝最深处那团黑下去的地方看过去。

不知道为什么，他忽然觉得那里面有什么东西也在“看”这一边。

不是看所有人。

像是先看见了他。

下一瞬，黑暗最深处，传来“当”的一声。

声音不大，却清得古怪。

不像石头撞石头，倒像一口埋了很多年的旧钟，被人在地底下，极轻地敲醒了一记。
