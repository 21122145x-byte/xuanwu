# 万界碑｜第二章开头 v2

黑。

先回来的不是痛，是黑。

像整个人一下被塞进了没有底的井里，四面都是冷硬的石，耳朵里却还嗡嗡作响，残留着上头塌陷时那阵碎石暴雨一样砸落的轰鸣。那声音远一下，近一下，像已经隔开了很厚一层土，又像还压在头顶，随时会再塌一次。

陆沉一开始连自己睁没睁眼都分不清。

嘴里全是灰。

苦，涩，还带着一点被石头磨过的腥甜，像牙龈哪儿破了，血正和烟尘一起黏在舌根上。鼻子里却是另一股味——不再是上面坑道里那种顶上来的浓重腥臭，而是更沉、更旧，像潮了很多年的石、烂在暗处的木，还有某种被压得太久、快要分不清是什么的陈败气，一层层闷在一起，贴着脸往里钻。

他想咳，胸口刚一抽，左边肩膀和后背就一齐炸开火辣辣的疼，疼得他眼前那团黑都跟着缩了一下。

这一下，他才真正醒过来。

自己还活着。

陆沉没急着动，只蜷在那里，先把那口要冲上来的咳嗽压了回去，硬生生吞进喉咙里。周围还有碎石在往下滚，窸窸窣窣，近一阵远一阵，像这地方还没彻底塌稳。再远一点，好像还有人声，可被土石和烟尘隔得太厚，传到这里已经闷得变了调，像隔着水在叫。

眼前还是黑。

可不是那种完全封死的黑。

更像烟尘还没散透，四周全蒙着一层厚灰，什么都在，偏偏什么都看不清。

陆沉撑了撑胳膊，想先从地上爬起来。

右手按下去的地方滑得发冷。

不像碎石。

也不像泥。

那触感太平，也太硬，冷得像一块埋了很多年、从来没见过天光的东西。

陆沉整个人猛地一僵，几乎是本能地把手缩了回来，肩背跟着往后一躲，后脑勺差点磕上后面的断岩。

这一躲牵得肩膀更疼，可他顾不上。

刚才掌心底下那一下太不对。

不是活物。

可也不像随处可见的碎石。

陆沉屏着气，在黑里盯着刚才按下去的地方，喉结慢慢滚了一下。烟尘还在缓缓地散，眼前依旧只有模模糊糊的一团暗影，分不清形状，也看不出高低，只觉得那地方比周围更沉一些，像黑里又压了一层黑。

跑。

脑子里第一个完整冒出来的念头，就是这个。

不管那是什么，先离远点。

陆沉咬着牙，先用膝盖和手肘一点点把自己往旁边撑开。地上全是碎石和断土，滑得厉害，稍一用力，腰侧和膝盖就被硌得生疼。他挪出去半尺，后背抵住一块凸出来的岩石，这才算勉强把自己从刚才按到那东西的地方移开。

人一离开，心口那股发紧倒没有散。

反而更清楚了。

不是因为看见了什么。

恰恰是因为还没看清。

烟尘又淡了一点。

他终于隐约看见，前面不远处有一片东西立在那里。

不是塌下来的乱石。

也不是坑壁。

更像一块从地里斜斜顶出来的硬面，边线太平，不像天然裂开的石头，黑里只露着模糊的一角，像有件本该死死埋在地下的旧物，被刚才那一下塌陷硬生生掀开了一点。

陆沉盯了两息，硬把目光挪开。

不能先看它。

先看自己还能不能走。

他低头摸了摸身上。左边肩膀疼得最狠，手肘、膝盖和腰侧都擦破了，掌心也还火辣辣地疼，但骨头应该没断。命还在，腿也还能使力。

这就够了。

上面的人未必死绝，瘦猴如果命大，说不定还在乱喊。疤脸那帮执鞭人更不可能就这么算了。这里如果有别的出口，他得先找路；要是没有，也得趁第二次塌下来之前，看看能不能往上爬回去。

陆沉扶着石壁，慢慢把自己撑起来。

脚下还虚，耳朵里的嗡鸣也没完全退，可总算能站住了。烟尘这时又散了一层，四周慢慢浮出一点轮廓：歪斜的断岩，半埋的木梁，脚边堆起来的碎石，还有前面那片始终沉着不动的暗影。

他刚想顺着石壁摸一圈，看看有没有能爬的地方，前面忽然传来一声极轻的响动。

当。

像有人隔着很远很远，在一块冷硬的石面上，用指节轻轻叩了一下。

陆沉全身一下绷住。

这回不是上面传下来的动静。

太近了。

近得像就从他脚底这片地里透出来，贴着石层，一路顶到骨头缝里。

他下意识往后退了半步，后背却已经贴上断岩，退无可退。

眼前那片暗影还是没动。

可那股安静忽然就变了味。

像不是死的。

像在等。

陆沉呼吸压得很低，视线却忍不住又落回那片东西上。

烟尘又散掉一点。

那东西的轮廓比刚才清楚了些。

果然不是乱石。

它立面平直，边线发冷，表面像覆着一层潮锈和泥垢，被岁月一寸寸糊死了，只在最上头露出一点模糊的棱。

还是什么都看不真切。

可只这点轮廓，已经足够让人心里发毛。

陆沉咬了咬牙，正想绕开它去别处找路，脚下的碎石忽然轻轻一松。

他心里一沉，刚要站稳，地面便又是一颤。

不大。

却很贼，像有什么东西在底下轻轻翻了一下身。

陆沉脚下一滑，整个人猛地朝前栽过去，手下意识往前一撑，掌心、手肘、半边肩膀全重重撞上那片冰冷的硬面。

冷。

不是石头被夜气浸出来的冷。

而是一种贴上去就直往骨头里钻的冷，像那东西根本不在地下，而是刚从更深、更阴的地方被谁拖上来的一截死物。

陆沉头皮一麻，几乎是立刻就想弹开。

可这一撞太实，他掌心已经顺着那片粗粝表面滑了半寸。指腹蹭过去，先是泥垢和潮锈一样的涩，接着又摸到一道很浅的凹痕。

不像天然裂纹。

倒像曾经被刻过什么，只是磨损得太厉害，早就辨不出原样了。

陆沉心里陡然一寒，猛地往后缩开，喘气一下都乱了。

他盯着那东西，胸口起伏得厉害，半晌没再上前。

烟尘还在慢慢散。

那东西也还只是露着一角模糊的轮廓。

可陆沉这时候已经明白，自己刚才不是单纯掉进了坑底。

他是掉到了某样东西旁边。

某样连矿场那些人见了，都恨不得立刻封死的东西。

而它现在，就立在他眼前。
