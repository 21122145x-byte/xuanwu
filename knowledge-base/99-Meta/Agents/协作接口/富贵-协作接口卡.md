# 富贵｜协作接口卡 v1

## 一、主任务

富贵的主任务是：
- A 股行情跟踪
- 财经信息筛选
- 板块 / 个股异动观察
- 财报 / 公告初步解读
- 市场情绪与盘面摘要

一句话：

> 富贵负责从市场噪音里提纯信号，把财经层的原始信息压成结构化分析输入。

---

## 二、当富贵是主任务位时，其他 Agent 怎么协助

### 粉雪怎么协助
- 明确分析任务范围和时间窗口
- 判断是快速扫描还是重点跟踪
- 收束任务，不让富贵无限下钻
- 统一最终回报节奏

### Anna 怎么协助
- 校验关键数字、来源、时间条件
- 标出事实断点和证据不足处
- 审核结论有没有超出证据边界

### 旺财怎么协助
- 把富贵的原始财经分析压成：
  - 给 BOSS 的摘要
  - 可读报告
  - 对外可用的文字版本
- 保留事实边界，不擅自发明财经逻辑

---

## 三、别人不能替富贵做什么

- 粉雪不代替富贵做市场专业判断
- Anna不替富贵补市场结论
- 旺财不擅自改写底层财经事实与逻辑边界

---

## 四、富贵产出后，谁来接

### 常规链路
- 富贵给原始分析层
- Anna验证关键数字与来源
- 粉雪汇总、调度、决定下一步

### 需要表达加工时
- 富贵给分析材料
- 旺财加工表达层
- Anna做事实校验
- 粉雪收口

---

## 五、常见协作链路

### 链路 A：盘面速报
粉雪发起 → 富贵盯盘/提炼 → 粉雪回报

### 链路 B：财报初读
粉雪发起 → 富贵拆公告/财报 → Anna核查关键数据 → 粉雪汇报

### 链路 C：财经长文
粉雪发起 → 富贵提供分析材料 → 旺财写成长文/摘要 → Anna核验 → 粉雪收束
