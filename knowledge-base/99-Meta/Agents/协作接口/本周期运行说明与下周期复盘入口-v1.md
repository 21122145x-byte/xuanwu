## 二、本周期要观察什么

### 1. 主任务位是否清楚
观察多 Agent 任务里，是否总能明确：
- 谁是主任务位
- 谁是协助位
- 有没有多人同时抢同一段工作

### 2. 任务发起是否仍然模糊
观察是否还频繁出现：
- 没有 task goal
- 没有 deliverable
- 没有 time requirement
- 边界没说清就开始干

### 3. handoff 是否真的被使用
观察交接时是否仍然出现：
- “你看一下”
- “差不多了”
- “做完了”

### 4. completion rules 是否真正生效
观察多 Agent 任务结束时，是否真的做到：
- 文件已落地
- 已知问题已暴露
- 下一步归属明确

### 5. 角色边界是否开始打架
重点看：
- 旺财是否越到调度 / 验证位
- 粉雪是否只分发不收束
- Anna 是否只驳回不提供修法
- 富贵是否结论过满或越出财经边界
