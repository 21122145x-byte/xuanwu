## 一、5层结构定义

推荐主链路如下：

1. **Constitution / Charter**：上位原则与边界  
2. **Program**：闭环机制、角色关系、流转与裁决结构  
3. **Project Plan**：本轮项目目标、阶段、责任、交付、验收  
4. **SOP / Runbook / Checklist**：具体环节操作说明  
5. **Prompt**：某个执行单元的调用说明

注意：

- **Program 是一个层，不是一份文件名。**
- 当前 Program 层内部已经拆成：
  1. **主协议**
  2. **规范卡**
  3. **接口卡**
- 后续索引与引用，不能再把这三类文档混写成“一个 Program 文档”。
