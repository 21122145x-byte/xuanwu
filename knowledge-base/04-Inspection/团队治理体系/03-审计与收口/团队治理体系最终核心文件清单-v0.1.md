## 五、当前不进入“最终核心清单”的文件类型

以下文件仍有价值，但当前不放入“最终核心文件清单”主列：

### 1. 当前版本实体件
例如：
- `团队协作宪法与治理总纲-v2.3-能力前置补丁版.md`
- `Program-团队协作运行主协议-v0.3-轻纠偏版.md`
- `Project-Plan规范说明及模板卡-v0.3.md`

原因：
- 它们属于内容源 / 对照源  
- 不是默认阅读入口

### 2. 历史版本件
例如：
- Charter v2.1 / v2.2  
- Plan v0.1 / v0.2  
- Prompt v0.1 / v0.2  
- SOP v0.1

原因：
- 主要用于回看、diff、回滚与复盘

### 3. 过渡术语件
- `Skill-MCP-Runtime-Pipeline-Harness关系卡-v0.1.md`

原因：
- 当前按过渡术语澄清件理解  
- 不再属于 Program 核心附件组

### 4. 过程性任务单 / 说明件
例如：
- `团队治理体系终版前整理任务单-v0.1.md`
- `团队治理体系命名口径说明-v0.1.md`
- `团队治理体系统一术语体系去向整理说明-v0.1.md`

原因：
- 有用，但不属于“默认最小交接包”
