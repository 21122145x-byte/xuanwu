## 模板本体

### 1. 最小单元名称
- [填写本次最小单元名称]

### 2. task goal
- [只写一个目标]

### 3. scope
- [写清本单元包含什么 / 不包含什么]

### 4. deliverable
- [只写一个可回收产物]

### 5. done condition
- [写清做到什么程度算完成]

### 6. constraints
- [写清边界、禁做项、格式要求、时间 / 成本约束]

### 7. return target
- [写清返回给谁 / 并入哪个主产物]

### 8. suggested model / cost tier
- [填写建议模型或成本档位；如未定，写“待主入口锁定”]

### 9. subagent prompt
```text
你是一次性生产 subagent。
你这次只处理一个最小单元，不做主编排，不扩题，不改阶段目标。

任务目标：
[填写 task goal]

范围：
[填写 scope]

唯一交付物：
[填写 deliverable]

完成标准：
[填写 done condition]

约束：
[填写 constraints]

回传目标：
[填写 return target]

提醒：
- 只做这一个单元
- 只交这一个产物
- 不追加长篇分析
- 完成后即回传并结束
```

### 10. 回收口径
- 回收检查 1：[产物是否单一]
- 回收检查 2：[边界是否未外溢]
- 回收检查 3：[是否满足 done condition]
- 回收检查 4：[是否可直接并入 return target]
