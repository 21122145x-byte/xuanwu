# 半自动任务看板模板 v1

- 适用协议：`/Users/xiaojingbo/.openclaw/workspace/knowledge-base/04-Inspection/2026-04-11-半自动任务执行协议_v1.md`
- 用途：用统一表头快速查看多条半自动任务的流转状态与停滞点

## 看板字段

- Task
- Current Status
- Owner
- Next Owner
- Next Action
- Updated At
- Risk / Blocker
- Artifact Where

## 示例

```md
- Task: Curie memory mainline
  - Current Status: review-ready
  - Owner: curie
  - Next Owner: anna
  - Next Action: audit
  - Updated At: 2026-04-11 01:30 Asia/Shanghai
  - Risk / Blocker: waiting audit handoff
  - Artifact Where: /path/mainline-v0.2.md

- Task: xxx
  - Current Status: audit-blocked
  - Owner: anna
  - Next Owner: curie
  - Next Action:补最小承接包
  - Updated At: 2026-04-11 01:40 Asia/Shanghai
  - Risk / Blocker: missing supporting artifacts
  - Artifact Where: /path/doc.md
```
