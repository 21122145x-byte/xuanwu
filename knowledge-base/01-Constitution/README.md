# 01-Constitution

> 团队制度与治理总则层。承载团队长期稳定规则、执行卡、审计意见、正式机制主线与其配套模板。

---

## 这一层放什么
- 团队协作总纲 / 上位法
- 固定主任务制度化文件
- 跨 Agent 协作协议与执行卡
- 团队级正式机制主线（如 Spawn）
- 团队模板链、审计意见、配套执行卡

## 这一层不放什么
- 项目专属方案与执行材料 → 去 `01-Projects/`
- 公共可复用方法论 → 去 `02-Areas/`
- 纯参考资料 / 外部文章 → 去 `03-Resources/`
- 阶段底稿 / 审计前准备卡 → 去 `03-Notes/`
- 元信息索引 / 状态页 / 维护规则 → 去 `99-Meta/`
