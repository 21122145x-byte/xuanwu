## 双层制度

- **稳定入口文档**：不带日期 / 版本号，用于长期引用、AGENTS 挂载、知识库导航。
- **下位过程文档**：带日期 / 版本号，用于草案、审计稿、施工稿、阶段稿。
- **更新规则**：下位文档验证完成后，回写并直接替换上位稳定入口；旧过程稿保留为轨迹，不再占正式入口名。
