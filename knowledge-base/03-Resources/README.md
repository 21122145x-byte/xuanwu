# 03-Resources

> 公共参考资料层。承载可查阅、可引用、可回溯的外部资料、经验记录、SOP 参考与研究辅助材料。

---

## 这一层放什么
- 协议、规则参考件
- 外部文章、研究资料、对比材料
- 工具 SOP、黄页、参考清单
- 经验教训与案例记录
- 写作研究、风格拆解等参考材料

## 这一层不放什么
- 当前运行状态结论 → 去 `99-Meta/`
- 项目正式方案、项目执行材料 → 去 `01-Projects/`
- 公共制度总纲 → 去 `01-Constitution/`
- 公共系统方法论正式件 → 去 `02-Areas/`
- 执行前准备卡 / 阶段底稿 → 去 `03-Notes/`

## 当前使用原则
- 资源层应以“可查阅、可引用”为主
- 如果某份资源已经被吸收到制度、项目或系统方法中，可继续保留原参考件，但不应把资源层当成正式结论层
- 资源层不是混合资料箱，应尽量通过 README / 索引维持导航

---

## 后续维护说明
- 新增参考件时，优先判断它是“资源”还是“正式方法 / 正式制度 / 项目正式件”
- Resources 中的重要材料，后续可在 Projects / Areas / Constitution 中做吸收，但资源原件仍可保留在本层
