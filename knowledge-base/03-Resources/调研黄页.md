## 工具速查

| 工具 | 用途 |
|------|------|
| `web_search` | 通用网页搜索（百度/Google/Kimi） |
| `web_fetch` | 抓取页面内容 |
| `xhs-explore` | 小红书内容搜索 |
| `xhs-publish` | 小红书内容发布 |
| `feishu_doc` | 飞书文档读写 |
| `feishu_wiki` | 飞书知识库 |
| `memory_search` | 长期记忆搜索 |
| `news-lite` | RSS 新闻聚合 |
| `faster-whisper` | 音视频转文字 |
| `tts` | 文字转语音 |

---

## 使用规则

1. **先查黄页**——收到调研任务，先看调研类型对应哪个工具
2. **多工具组合**——复杂调研可能需要多个工具配合
3. **记录来源**——调研结论要标注参考来源
4. **黄页迭代**——发现新工具/渠道，及时更新
