# Writer-Agent 反馈卡模板

## 用途

用于把抽象反馈转成可执行修订动作，并能写回 learning memory。

---

## 反馈卡

- **对应任务**：
- **问题定位**：
- **错误类型**：`结构 / 人物 / 节奏 / 语言 / 设定`
- **具体表现**：
- **为什么不成立**：
- **修正原则**：
- **优先级**：`P0 / P1 / P2`
- **是否写回 Memory**：`是 / 否`
- **写回层级**：`identity / project / learning / feedback`

---

## 填写原则

1. `问题定位` 必须定位到段落 / 场景 / 功能层
2. `为什么不成立` 不能写“感觉不对”
3. `修正原则` 必须能指导下一次改稿
4. 只有可复用经验才写回 `learning` 或 `identity`

---

## 示例

- **对应任务**：第一章开头修订
- **问题定位**：前两段
- **错误类型**：`节奏`
- **具体表现**：解释感偏重，人物进入太慢
- **为什么不成立**：开头先讲系统信息，读者还没先抓住黎照这个人
- **修正原则**：开头优先给人物压力与动作，再让世界信息从岗位环境漏出
- **优先级**：`P0`
- **是否写回 Memory**：`是`
- **写回层级**：`learning`
