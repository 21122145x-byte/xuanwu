# Writer-Agent 基础 Checklist

## 用途

用于第 1 周和第 2 周的最小评估，判断一轮输出是否达到基本可用。

---

## A. 任务层

- [ ] 任务类型已明确
- [ ] 任务目标只有一个核心重点
- [ ] 交付物具体可判断
- [ ] 本次禁忌已写明

---

## B. 记忆层

- [ ] 已读取 Identity 相关记忆
- [ ] 已读取 Project 相关记忆
- [ ] 已读取 Recent Learning
- [ ] 没有把临时任务混进长期记忆

---

## C. 结构层

- [ ] 已先出骨架，不是直接冲正文
- [ ] 一句话梗概清楚
- [ ] 场景 / 章节功能清楚
- [ ] 结尾有拉力

---

## D. 文本层

- [ ] 人物先于设定出现
- [ ] 没有明显整段解释
- [ ] 至少有一个成立的动作 / 画面 / 选择
- [ ] 没有明显重复段落

---

## E. 判断层

- [ ] 已跑自审五问
- [ ] 已标出错误类型
- [ ] 已给出删改优先级
- [ ] 至少提炼 1 条修正原则或 learning

---

## 结论

- [ ] 通过：进入下一阶段
- [ ] 需修改：局部修订
- [ ] 打回：重搭骨架 / 重写
