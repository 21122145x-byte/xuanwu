# Writer-Agent 任务卡模板

## 用途

用于定义当前任务，避免 agent 一上来就直接写正文。

---

## 任务卡

- **任务名称**：
- **任务类型**：`research / outline / draft / revision / self_review`
- **所属项目**：
- **任务目标**：
- **本次交付物**：
- **输入材料**：
- **本次重点**：
- **本次禁忌**：
- **成功标准**：
- **截止时间**：
- **当前状态**：`待开始 / 进行中 / 待审核 / 已完成`

---

## 填写原则

1. `任务目标` 只写本轮最重要的一件事
2. `本次交付物` 必须具体，不能写“优化一下”
3. `本次禁忌` 至少写 1 条
4. `成功标准` 必须可判断

---

## 示例

- **任务名称**：第一章开头重写
- **任务类型**：`revision`
- **所属项目**：`SaltFire-Project`
- **任务目标**：让黎照先作为人出现，而不是先堆设定
- **本次交付物**：第一章开头 800–1200 字修订稿
- **输入材料**：旧稿、开头修订原则、人物设定卡
- **本次重点**：人物压力、抓力、陌生词减量
- **本次禁忌**：不要整段说明世界观
- **成功标准**：前两段内立住人物状态，并形成继续读下去的拉力
- **截止时间**：2026-03-18
- **当前状态**：`进行中`
