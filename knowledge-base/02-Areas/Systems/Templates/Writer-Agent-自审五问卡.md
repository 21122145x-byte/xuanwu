# Writer-Agent 自审五问卡

## 用途

用于正文完成后做最小但有效的 Judgment，自查是否成立。

---

## 自审五问

1. **这段真正推进了什么？**
   - 推情节 / 立人物 / 埋信息 / 提升风险 / 转换关系

2. **最有力的点是什么？**
   - 哪个动作、画面、选择、句子最成立

3. **最虚的地方是什么？**
   - 哪一段在解释、发飘、硬抒情、假热闹

4. **哪句最像作者在用力？**
   - 找出最像“想写好”的那句，而不是最自然的那句

5. **如果删 40%，先删哪里？**
   - 优先删：重复说明、无功能段落、假钩子、硬解释

---

## 输出格式

- **推进功能**：
- **最有力点**：
- **最虚点**：
- **最用力句**：
- **优先删改处**：
- **结论**：`可进入下一阶段 / 需重写 / 需局部修订`

---

## 原则

- 不做空泛夸奖
- 不写“整体还行”这种废话
- 每次自审至少指出 1 个真实问题
