# Systems

> 公共系统方法与系统级知识区。承载可复用的系统设计、架构方法、运行经验、模板体系与专题性系统方案。

---

## 这一层放什么
- 可复用的系统设计与架构方案
- Memory / MemOS / Agent 等系统方法论
- 系统级 SOP、模板、Checklist
- 多项目可复用的执行框架
- 已经超出单项目边界的公共系统知识

## 这一层不放什么
- 某个项目专属方案与推进记录 → 去 `01-Projects/`
- 纯参考资料 / 文章收藏 → 去 `03-Resources/`
- 临时执行件 / 待审中间稿 → 去 `03-Notes/`
- 团队制度总纲 / 宪法 → 去 `01-Constitution/`
